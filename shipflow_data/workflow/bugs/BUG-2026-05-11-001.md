# Bug Record: BUG-2026-05-11-001

## Summary

- Title: Vercel refuses Flutter app build because vercel.json is invalid JSON
- Current status: `fix-attempted`
- Severity: `high`
- Owner: Diane
- Next command: `/sf-test --retest BUG-2026-05-11-001`

## Reproduction

- Preconditions:
  - Vercel project uses `tubeflow_app/vercel.json` for the Flutter web app deployment.
- Steps:
  1. Trigger a Vercel deployment for the Flutter app.
  2. Vercel reads `vercel.json`.

## Expected Behavior

- Vercel parses the project configuration, installs Flutter, and starts `bash build.sh`.

## Observed Behavior

- Vercel refuses the deployment before build logs are available because `vercel.json` is invalid.

## Evidence

- `jq . tubeflow_app/vercel.json` failed with `parse error: Unmatched ']' at line 21, column 7`.
- Vercel docs confirm `buildCommand`, `installCommand`, `outputDirectory`, `framework`, `headers`, and `rewrites` are valid `vercel.json` fields.
